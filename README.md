# 製薬DXジャーナル

## テーマ
製薬業界におけるデジタルトランスフォーメーション（DX）の最新動向と実践事例を発信するブログ。AI創薬、DCT（分散型臨床試験）、RWD活用など、製薬バリューチェーン全体のデジタル化を追う。

## ターゲット読者
- 製薬企業のDX推進・IT部門担当者
- 臨床開発のデジタル化に関心のあるプロジェクトリーダー
- ヘルステック・製薬テックに関わるエンジニア
